<h1>Projeto Pixels Art</h1>

<p>Este projeto foi desenvolvido durante o curso de Desenvolvimento Web da Trybe para consolidação do aprendizado em JavaScript, como manipulação de arrays e objetos, DOM, escutadores de eventos.</p>
<p>A proposta do projeto era desenvolver um editor de arte com pixels, no qual a pessoa usuária pode escolher a cor em uma paleta de cores e pintar os quadrados brancos.</p>
<p>Além disso, ao pressionar o botão <em><b>"Cores aleatórias"</b></em> a paleta de cores muda para cores que são sorteadas aleatoriamente, e o botão <em><b>"Limpar"</b></em> faz com que todos os quadrados voltem a ser brancos.</p>
