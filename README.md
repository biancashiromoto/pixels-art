<h1>🇧🇷 Projeto Pixels Art 🎨🖌</h1>

<p>Este projeto foi desenvolvido durante o curso de Desenvolvimento Web da Trybe para consolidação do aprendizado em JavaScript, como manipulação de arrays e objetos, DOM, escutadores de eventos.</p>
<p>A proposta do projeto era desenvolver um editor de arte simples, no qual a pessoa usuária pode escolher a cor em uma paleta de cores e pintar os quadrados brancos.</p>
<p>A paleta de cores inicial é selecionada aleatoriamente, e pode ser redefinida de também ao pressionar o botão <em><b>"Random colors"</b></em>, e o botão <em><b>"Clear board"</b></em> faz com que todos os quadrados voltem a ser brancos.</p>
