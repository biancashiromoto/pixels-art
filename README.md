<h1 style="text-align:center;font-size:2.5rem;font-weight:600">Pixels Art 🎨🖌</h1>

<span style="font-size:1.25rem; display:flex; justify-content:center">🇧🇷</span>
<p style="font-size:1.25rem" style="font-size:1.25rem">Este projeto foi desenvolvido durante o curso de Desenvolvimento Web da Trybe para consolidação do aprendizado em JavaScript, como manipulação de arrays e objetos, DOM e escutadores de eventos.</p>
<p style="font-size:1.25rem">A proposta do projeto era desenvolver um editor de arte simples, no qual a pessoa usuária pode escolher a cor em uma paleta de cores e pintar os quadrados brancos.</p>
<p style="font-size:1.25rem">A paleta de cores inicial é selecionada aleatoriamente, e pode ser redefinida ao pressionar o botão <em><b>"Random colors"</b></em>, e o botão <em><b>"Clear board"</b></em> faz com que todos os quadrados voltem a ser brancos.</p>
<p style="font-size:1.25rem">No futuro pretendo aplicar conhecimentos em localStorage, para que a paleta de cores e os quadrados pintados fiquem salvos ao recarregar a página, e adicionar uma opção para que o usuário possa selecionar um número que seja o tamanho do quadro.</p>
<br>
<hr>
<span style="font-size:1.25rem; display:flex; justify-content:center">🇺🇸</span>
<p style="font-size:1.25rem">This project was developed during Trybe's Web Development course to consolidate learning in JavaScript, as array and object manipulation, DOM and event listeners.</p>
<p style="font-size:1.25rem">The project's proposal was to develop a simple art editor, in wich the user is able to choose a color from a palette and paint the white squares.</p>
<p style="font-size:1.25rem">The initial color palette is randomly selected, and can be redefined when pressing the <em><b>"Random colors"</b></em> button, and the <em><b>"Clear board"</b></em> button makes all the squares colors white.</p>
<p style="font-size:1.25rem">In the future I plan to apply localStorage knowledge to save the color palette and the painted squares when the page is refreshed, and add an option so the user can select a number of squares to set the board size.</p>
